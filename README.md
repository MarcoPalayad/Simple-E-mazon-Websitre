# Simple Emazon Website

A beginner-friendly e-commerce project that demonstrates core web development concepts. This educational project mimics basic Amazon functionality including product browsing, shopping cart, checkout, and order tracking.

## Features

- Product catalog with images and pricing
- Shopping cart functionality
- Checkout process
- Order tracking
- Responsive design for all devices

## Project Structure

```
├── amazon.html          # Main product page
├── checkout.html        # Checkout page
├── orders.html          # Orders page
├── tracking.html        # Tracking page
├── backend/products.json # Product database
├── data/                # Cart and product logic
├── styles/              # CSS files
└── images/              # Assets
```

## Technologies

- HTML5, CSS3, Vanilla JavaScript
- JSON for product storage
- Responsive CSS layouts

## Getting Started

1. Clone or download this repository
2. Open `amazon.html` in your browser
3. Start shopping!

## Learning Concepts

- DOM manipulation with JavaScript
- Event handling and listeners
- Local storage for cart persistence
- JSON data handling
- Responsive web design
- CSS Grid and Flexbox
